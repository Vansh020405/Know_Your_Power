# Know Your Power

A mobile-first web app to help users understand their rights, eligibility for benefits, and risks in contracts.

## Features

1.  **Authority Power Checker**: Check if an authority (Police, HR, Landlord) can legally force you to do something.
2.  **Eligibility Checker**: Find government schemes and benefits based on your profile.
3.  **Document Decoder**: Paste contract text to find risky clauses (Lock-in periods, etc.).

## Tech Stack

-   React (Vite)
-   Vanilla CSS (Mobile-first design)
-   Lucide React (Icons)
-   React Router DOM (Navigation)

## How to Run

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the application (Frontend + Backend):
    ```bash
    npm start
    ```

3.  Open the link shown in the terminal (usually `http://localhost:5173`) on your mobile or desktop (resize to mobile view for best experience).
